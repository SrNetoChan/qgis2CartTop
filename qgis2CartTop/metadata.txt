      para a Cartografia Vectorial e de Imagem" (CartTop / ReCart) from
      Direcção-Geral do Território (Portugal). Also contains tools for eficient 
      editing and improve data consistency.

      (PT) Plugin QGIS para ajudar na importação de dados numa base de dados
      PostgreSQL/PostGIS segundo o modelo de dados definido pelas "Normas e
      Especificações Técnicas para a Cartografia Vectorial e de Imagem" 
      (CartTop / ReCart) da Direcção-Geral do Território (Portugal). Contém
      também ferramentas para uma edição mais eficiente e melhoria da
      consistência dos dados.
